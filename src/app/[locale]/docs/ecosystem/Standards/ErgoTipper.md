# ErgoTipper & Rewards Bots

Reward community members with instant tipping — on Telegram, Discord, Reddit, and Twitter.

## Getting Started

- DM or mention the ErgoTipperBot to create your tip wallet.
- Fund your wallet with ERG or supported tokens.
- Use simple commands to tip others — if the recipient has no wallet, one is auto-created.
- Restore your tip wallet across platforms with the same seed phrase.

## Supported Platforms

- Reddit, Discord, Telegram, Twitter

## Security Note

Never use your tip wallet as your main wallet. It’s custodial and not 100% secure.

## How to Add a Token

Submit a PR to the bot’s repo. For NFTs, use the separate NFT list.

# Community Administration

Create an open, safe, and engaged community around your project.

## Essentials

- **Anti-Spam/Scam:** Enable bots (OrgRobot, GrepRobot, Wickbot) in Telegram; Automod, Wickbot, or Dyno in Discord.
- **Moderation Rules:** Block common scam words, suspicious links, excessive mentions, and inappropriate emojis.
- **User Onboarding:** Greet new users, verify real people, answer common questions.
- **Education:** Regularly remind users about scams, best practices, and security.
- **Chat Bridging:** Link Telegram and Discord for unified support (see [Chat Bridge](./Chat-Bridge.md)).

## Moderation Tips

- Disable threads if not used (Discord)
- Exclude active members from strict spam rules
- Regular review of automod/ban lists

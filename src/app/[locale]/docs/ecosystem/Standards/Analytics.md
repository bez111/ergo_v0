# Analytics & Developer Contributions

Make your project visible and contribute to the ecosystem’s growth.

## For Analytics

- Integrate on-chain metrics via [ergo.watch](https://ergo.watch), [DeFiLlama](https://defillama.com/chain/Ergo), [Artemis](https://artemis.xyz)
- Track usage, active addresses, liquidity, TVL, etc.
- Keep your project up to date in ecosystem aggregators (ergcube, sigmaverse, weekly dev reports)

## To Get Featured

- Add your GitHub org/repo to [ecosystem-devs list](#)
- Fill out the onboarding form or make a PR to ecosystem docs

Showcase your impact, reach new users, and help grow the Ergo network!

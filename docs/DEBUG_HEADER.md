# 🔍 ОТЛАДКА ПРОБЛЕМЫ С ХЕДЕРОМ

## Проблема:
Текст "ERGO" в хедере отображается белым вместо оранжевого.

## Что нужно проверить в браузере:

### 1. Откройте DevTools (F12)

### 2. Найдите элемент с текстом "ERGO"

### 3. Проверьте в панели Styles:
- Есть ли класс `text-primary`?
- Какое значение у CSS переменной `--primary`?
- Применяется ли цвет `color: hsl(var(--primary))`?

### 4. В консоли браузера выполните:
```javascript
getComputedStyle(document.documentElement).getPropertyValue('--primary')
```

### 5. Проверьте вкладку Network:
- Загружается ли файл globals.css?
- Есть ли ошибки 404?

## Ожидаемые значения:
- `--primary: 25 95% 53%` (это оранжевый цвет)
- Класс `text-primary` должен применять этот цвет

## Возможные причины:
1. CSS переменная не определена
2. Tailwind не компилирует класс `text-primary`
3. Другие стили перебивают цвет
4. Проблема с PostCSS/Tailwind конфигурацией

Сообщите мне, что вы видите в DevTools!

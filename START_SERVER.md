# Инструкции по запуску сервера

## Проблема
Сервер не запускается из-за отсутствующих зависимостей Next.js.

## Решение

1. **Откройте терминал Mac** (не в Cursor, а отдельное приложение Terminal)

2. **Перейдите в папку проекта:**
   ```bash
   cd ~/Desktop/4
   ```

3. **Удалите старые файлы (если есть):**
   ```bash
   rm -rf node_modules package-lock.json
   ```

4. **Установите все зависимости:**
   ```bash
   npm install
   ```

   Если появятся предупреждения о версии Node.js, установите Node.js версии 20 или выше:
   ```bash
   # Установка через Homebrew
   brew install node@20
   ```

5. **Запустите сервер на порту 3001:**
   ```bash
   npm run dev -- -p 3001
   ```

   Или альтернативно:
   ```bash
   npx next dev -p 3001
   ```

## Проверка

После успешного запуска вы увидите:
```
▲ Next.js 15.5.2
- Local:        http://localhost:3001
- Network:      http://192.168.15.90:3001
✓ Ready
```

Откройте http://localhost:3001 в браузере.

## Если проблемы продолжаются

1. Убедитесь, что у вас Node.js версии 18 или выше:
   ```bash
   node --version
   ```

2. Попробуйте установить с флагом legacy-peer-deps:
   ```bash
   npm install --legacy-peer-deps
   ```

3. Проверьте, не занят ли порт 3001:
   ```bash
   lsof -i :3001
   ```

## Альтернативный способ

Я создал скрипт `check-and-start.sh`, который можно запустить:
```bash
./check-and-start.sh
```

Он автоматически проверит состояние и запустит сервер.

# Chat Bridge Setup & Best Practices

Unite your community across Telegram and Discord with seamless chat bridges.

## How to Set Up

1. Invite `BridgeBot#9505` to your Discord server.
2. Add ErgoBridgeBot to your Telegram group (use `/allowInvitingBots` if Shieldy is enabled).
3. Notify the ecosystem bridge admin (e.g., @Glasgow) to link your channels.
4. For custom bots, provide the token ID to the bridge admin.

## Known Limitations

- Discord bots can’t reply when spoofing.
- Telegram deletions aren’t mirrored to Discord.
- Profile photo sync is manual.
- Best practice: restrict new Telegram users until verification.

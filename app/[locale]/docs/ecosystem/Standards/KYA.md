# KYA (Know Your Assumptions) Standard

Promote transparency and user awareness in DeFi.

## What is KYA?

A framework to make all protocol and security assumptions explicit for every dApp, wallet, or service.

## KYA Prompts Cover

- Security assumptions (cryptography, consensus)
- Economic assumptions (incentives, tokenomics)
- User responsibilities (key management, risks)
- External dependencies (oracles, APIs)
- Transparency (centralized components)

## How to Add KYA Prompts

- Show a clear prompt before users interact with your protocol (example: “Using this dApp requires trusting the following oracles and assumes users keep their keys secure…”)
- Link to detailed docs or KYA section

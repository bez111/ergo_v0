## 🔄 Процесс рефакторинга

1. **Создать ветку рефакторинга**
```bash
git checkout -b refactor/cleanup-unused-components
```

2. **Удалить неиспользуемые компоненты**
```bash
rm components/ui/breadcrumbs.tsx
rm components/ui/grid.tsx
rm components/ui/page-container.tsx
```

3. **Обновить экспорты**
```bash
# Обновить components/ui/index.ts
```

4. **Проверить сборку**
```bash
npm run build
npm run test
```

5. **Измерить улучшения**
```bash
# Проверить размер бандла
npm run analyze
```
